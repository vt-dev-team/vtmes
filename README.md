# Vegetable消息系统

语言：简体中文 | [English](#vegetable-message-system)

## 安装

1. 安装必备软件

  1. [git](https://git-scm.com/)
  2. [nodejs](https://nodejs.org/)

2. 安装依赖

```bash
git clone https://github.com/vt-dev-team/vtmes
cd vtmes
yarn
yarn add nodejs-websocket -S
```

3. 运行前端和后端

```bash
#前端
yarn dev
#后端
cd backend
node server.js
```
